Đề tự tạo

Đề do Admin tạo/chia sẻ sẽ hiện cho mọi người dùng; đề bạn tự tạo (không tick chia sẻ) chỉ riêng mình bạn thấy

Bạn chưa tạo đề nào. Hãy tạo đề mới từ file JSON hoặc file PDF bên dưới.

Tạo đề từ file JSON

Nhập file questions.json (định dạng chuẩn: id/question/options/answer). Đề sẽ được lưu riêng trong mục "Đề tự tạo", không ảnh hưởng tới 300 câu hỏi gốc.

Kéo thả hoặc bấm để chọn file questions.json

Hệ thống sẽ tự kiểm tra định dạng JSON hợp lệ

Tạo đề từ file PDF + đáp án Excel

Upload đề thi dạng PDF, tải file mẫu Excel để điền đáp án rồi upload lại. Khi làm bài, hệ thống bấm giờ tổng cho toàn bài (1 phút 30 giây / câu, tính chung cho cả đề).

Kéo thả hoặc bấm để chọn file PDF đề thi

Chỉ chấp nhận định dạng .pdf

Bước 2: Đáp án

Kéo thả hoặc bấm để chọn file Excel đáp án đã điền

Định dạng .xlsx / .xls, gồm cột STT và Đáp án

Xuất PDF

Giới hạn kỹ thuật: Với ràng buộc chỉ dùng Chart.js/Font Awesome/Google Fonts (không jsPDF), việc tạo file PDF nhị phân thật là không khả thi thuần JS. Giải pháp thay thế: dùng chức năng In (Print) của trình duyệt để lưu thành PDF, đảm bảo layout đẹp khi in.
